# 🐧 Linux CLI Fundamentals - OverTheWire Bandit Labs

Este repositório contém minhas anotações práticas, documentação técnica e aprendizados durante a resolução dos desafios do **OverTheWire: Bandit**. 

O objetivo deste projeto é consolidar fundamentos práticos de administração de sistemas Linux, navegação via CLI, manipulação de arquivos, redes e segurança.

---

## 🎯 Objetivos de Aprendizado
-  Conexão remota via **SSH** em portas não-padrão (Level 0)
-  Navegação e leitura de arquivos ocultos ou com caracteres especiais
-  Manipulação e filtragem avançada de dados (`grep`, `find`, `sort`, `uniq`)
-  Análise de permissões de arquivo e privilégios (`chmod`, `chown`, `SUID`)
-  Inspeção de tráfego e conexões de rede (`netcat`, `openssl`)
-  Automação básica de tarefas com **Bash Shell Scripting**
